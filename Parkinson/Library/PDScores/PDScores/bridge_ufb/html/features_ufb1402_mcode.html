1% Receives raw features, normalization parameters and feature weights, and
 2% computes user feedback value.
 3% Inputs:
 4%  ftvec - feature vector (1xN)
 5%  wvec  - weight vector (Nx1)
 6%  ilog  - list of features to apply logarithm (variance stabilization)
 7%  ftmin - feature normalization vector minimum (1xN)
 8%  ftmax - feature normalization vector maximum (1xN)
 9%  fbmin - user feedback minimum
10%  fbmax - user feedback maximum
11% Outputs:
12%  fbnorm - normalized user feedback value
13%
14% (CC BY-SA 3.0) Max Little, 2014
15function fbnorm = features_ufb(ftvec,wvec,ilog,ftmin,ftmax,fbmin,fbmax)
16
17ftvecnorm = ftvec;
18ftvecnorm(:,ilog) = log10(ftvecnorm(:,ilog));
19ftvecnorm = 2*((ftvecnorm-ftmin)./(ftmax-ftmin))-1;
20
21% Projection of features onto 1st PC to create simple user feedback
22fb = ftvecnorm*wvec;
23
24% Normalize and re-scale 0-100, clamp
25fbnorm = max(min((fb-fbmin)/(fbmax-fbmin)*100,100),0);
